Saving a book

You can also send a POST request to:



to add a book to the user's database. This requires the posting of a body with the following JSON format:

{ "book": { "title": "Title", "id": "ID", "authors": [ "author name" ], "genres" : [ "genre" ], "imageLink": "link", "start": "2024-12-30", "end": "-", "status": "Reading" } }


  • title: specifies the book title.
  • id: specifies the book id (typically the ISBN number).
  • authors: an array of authors of the book.
  • genres: consists of an array of genres that describe the book.
  • imageLink: a URL to the image to use as the book cover.
  • start: specifies the start date of reading the book. Leave as "-" if no date is to be specified.
  • end: specifies the end date of reading the book. Leave as "-" if no date is to be specified.
  • status: describes the user's current status with the book. Takes the values "To Read", "Reading", and "Read".
